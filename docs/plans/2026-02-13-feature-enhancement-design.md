## 一、计时器增强

### 1.1 自定义时间输入

**控件**: 滚轮选择器（CupertinoPicker 风格）

| 滚轮 | 范围 | 说明 |
|------|------|------|
| 分钟 | 0-5 | 分钟选择 |
| 秒 | 00/10/20/30/40/50 | 每10秒一格 |

**交互**:
- 在运动开始前设置休息时长
- 运动期间不可修改休息时长
- 默认值: 60秒（1分钟）

### 1.2 间歇训练模式

**核心逻辑**:
- 运动阶段：正向计时（从0开始累加）
- 休息阶段：倒计时（使用预设时长）
- 休息结束：自动切换回运动阶段

**状态流转**:

```
┌─────────────────────────────────────────────────────────────┐
│                        状态机                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [空闲] ──设置休息时长──> [准备开始]                         │
│                            │                                │
│                        开始运动                              │
│                            ↓                                │
│                      [运动中] ←─────────────┐               │
│                      正向计时                │               │
│                      显示"第N组"            │               │
│                            │                │               │
│                      开始休息                │ 休息结束       │
│                      (可暂停)                │ (自动)        │
│                            ↓                │               │
│                      [休息中] ───────────────┘               │
│                      倒计时                                  │
│                      显示"已完成N组"                         │
│                            │                                │
│                      结束运动                                │
│                            ↓                                │
│                      [已完成]                                │
│                      显示"本次训练完成N组"                    │
│                            │                                │
│                    保存 / 继续运动                           │
│                            │                                │
│               ┌────────────┴────────────┐                   │
│               ↓                         ↓                   │
│          [已保存]               [继续运动]                   │
│          记录数据               恢复到运动中                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**状态显示规则**:

| 状态 | 屏幕显示 | 说明 |
|------|----------|------|
| 运动中（第N组开始）| "正在进行第N组" | N从1开始 |
| 休息中 | "已完成N组，准备进行第N+1组" | N为刚完成的组数 |
| 休息结束自动切换 | "正在进行第N+1组" | 自动进入下一组 |
| 结束运动 | "本次训练已完成N组" | 显示总组数 |

**按钮配置**:

| 状态 | 主按钮 | 次按钮 |
|------|--------|--------|
| 运动中 | 开始休息 | 暂停 / 结束运动 |
| 运动暂停 | 继续 | 结束运动 |
| 休息中 | 跳过休息 | - |
| 已完成 | 保存 | 继续运动 |

### 1.3 数据模型扩展

```dart
class WorkoutSession {
  final String id;
  final int totalSets;         // 完成组数
  final int totalWorkTimeMs;   // 运动总时长（毫秒）
  final int totalRestTimeMs;   // 休息总时长（毫秒）
  final DateTime startTime;    // 训练开始时间
  final DateTime? endTime;     // 训练结束时间（null表示进行中）
  final String? notes;         // 备注（可选）
}
```

### 2.2 统计数据计算

**本周/本月汇总**:

| 指标 | 计算方式 |
|------|----------|
| 总组数 | `SUM(totalSets)` WHERE startTime 在时间范围 |
| 总时长 | `SUM(totalWorkTimeMs + totalRestTimeMs)` |
| 训练天数 | `COUNT(DISTINCT DATE(startTime))` |

**个人最佳**:

| 记录 | 计算方式 |
|------|----------|
| 单次最多组数 | `MAX(totalSets)` |
| 单次最长训练 | `MAX(totalWorkTimeMs + totalRestTimeMs)` |
| 连续训练天数 | 最长连续日期序列 |

### 2.3 数据库扩展

**新增索引**:
```sql
CREATE INDEX idx_start_time ON workout_sessions(start_time);
CREATE INDEX idx_end_time ON workout_sessions(end_time);
```

## 五、技术要点

### 5.1 状态管理

新增 `TrainingProvider` 管理训练状态:
```dart
enum TrainingState {
  idle,           // 空闲
  ready,          // 准备开始（已设置休息时长）
  exercising,     // 运动中
  exercisePaused, // 运动暂停
  resting,        // 休息中
  completed,      // 已完成
}
```

### 5.2 计时器实现

- 运动计时：`Stopwatch` + `Timer.periodic` 更新UI
- 休息计时：`Timer.periodic` 倒计时
- 后台运行：Android Foreground Service（现有）

### 5.3 主题切换

- 使用 `Provider` + `ChangeNotifier` 管理主题状态
- 切换时保存到 `SharedPreferences`
- 启动时读取并应用
